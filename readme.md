The dependecies for this projects can be installed using.

```javascript

```

# or

```javascript
    yarn install
```

#### For iOS (with Expo)


```javascript
    yarn ios
```

### For Android (with Expo)

```javascript

```

# or

```javascript
    yarn android
```
